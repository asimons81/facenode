# Examples

This directory contains runnable examples and sample configurations.

| Directory | Description |
|-----------|-------------|
| `mock-demo/` | Standalone WebSocket server, bridge walkthrough, and config preset for developing the avatar without a live Hermes instance. |
| `mock-demo/config.example.json` | Example dashboard config preset for the mock demo flow. |

> Examples are added as each phase ships. See [ROADMAP.md](../ROADMAP.md) for the timeline.
